# 🎓 Student Burnout Dashboard

An AI-powered web application that predicts a student's burnout risk based on study habits and lifestyle factors. The application uses a Machine Learning model to analyze user inputs and provides a burnout prediction, confidence score, personalized recommendations, and visual analytics.

## 🚀 Features

- Predicts student burnout risk (Low, Medium, High)
- Displays prediction confidence
- Personalized recommendations
- Interactive dashboard
- Bar Chart
- Pie Chart
- Line Chart
- Light/Dark Mode
- Responsive UI

